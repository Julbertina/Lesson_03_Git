Commands

Cписок команд, которые мы рассматривали на занятии по работе с терминалом
  1. git config --global user.name "[имя]"
  2. git config --global user.email "[адрес электронной почты]"
  3. git init [название проекта]
  4. git status
  5. git add [файл]
  6. git commit -m "[сообщение с описанием]"
  7. git log
  8. git reset [файл]
  9. git push --set-upstream origin master
